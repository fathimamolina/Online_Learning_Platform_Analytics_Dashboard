# Online Learning Platform Analytics Dashboard

## Project Overview
This project analyzes online learning platform data using Power BI. The dashboard provides insights into course subscribers, reviews, pricing, and subject performance.

## Tools Used
- Power BI Desktop
- Microsoft Excel
- Kaggle Dataset

## Features
- KPI Cards
- Subject-wise Subscriber Analysis
- Top 10 Courses Visualization
- Price vs Subscribers Analysis
- Interactive Subject Filter

## Dataset
Dataset sourced from Kaggle - Udemy Courses Dataset.

- raw_dataset.xlsx
- cleaned_dataset.csv
